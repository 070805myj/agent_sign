# 作业车间调度优化智能体 (JSP Scheduling Agent)

基于改进遗传算法的作业车间调度问题（Job-Shop Scheduling Problem, JSP）求解系统，支持基线算法对比和可视化分析。

## 核心特性

- **遗传算法（GA）**：采用 IPOX 交叉 + 工件块移位变异，支持自适应变异率和精英保留
- **基线算法**：FCFS（先来先服务）和 SPT（最短加工时间优先）串行实现
- **贪婪解码器**：标准左移贪婪解码，O(nm) 时间复杂度
- **可视化分析**：甘特图、收敛曲线、算法对比柱状图
- **KPI 指标**：Makespan、设备利用率、负载均衡度等多维度评估
- **模块化 GUI**：基于 Tkinter 的分层架构（框架/组件/服务）

## 快速开始

### 环境要求

- Python 3.8+
- Windows / macOS / Linux

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

或双击 `start_app.bat`（Windows）

### 使用流程

1. **数据配置**：选择内置测试用例（FT06/FT10等）或自定义问题规模
2. **参数设置**：调整种群大小、迭代次数、交叉/变异概率
3. **运行算法**：依次运行 GA 优化、FCFS 基线、SPT 基线
4. **结果分析**：查看 KPI 指标、调度方案详情、生成图表

## 算法说明

### 遗传算法算子

| 算子 | 方法 | 说明 |
|------|------|------|
| 交叉 | IPOX | 改进的保持优先级交叉，保留优秀工序结构 |
| 变异 | 工件块移位 | 随机选择工件块插入新位置，自适应变异率 |
| 选择 | 截断选择 | 保留适应度最高的前50%个体 |
| 保留 | 精英策略 | 最优个体直接进入下一代 |

### 解码算法

采用标准左移贪婪解码（Left-shift Greedy Decoding），将工序序列转换为时间表：

```
s(j,o) = max( 机器可用时间, 工件前序完工时间 )
c(j,o) = s(j,o) + p(j,o)
```

### KPI 指标

| 指标 | 公式 | 说明 |
|------|------|------|
| Makespan | C_max = max(c_{j,n-1}) | 最大完工时间 |
| 设备利用率 | U = Σp / (m·C_max) | 机器平均负荷 |
| 负载方差 | σ² = (1/m)Σ(W_k - W̄)² | 负载均衡度 |
| 优化率 | R = (C_base - C_GA) / C_base | 相对基线改进 |
